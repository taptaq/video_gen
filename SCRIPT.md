# 润滑液科普短视频脚本

适用画幅: 9:16  
建议时长: 42 秒左右  
定位: 抖音信息流科普 / 亲密健康入门向

## 口播版

00:00 - 00:04  
润滑液，真的不只是“救急用品”。

00:04 - 00:10  
很多人一提到它，就会联想到尴尬。其实更准确的理解是，它是一个让体验更顺、更舒服的工具。

00:10 - 00:16  
因为压力、节奏、环境变化，甚至只是当天状态不同，自然润滑都可能不一样。所以“会用”，不代表“有问题”。

00:16 - 00:24  
润滑液的发展史，本质上就是一部从“先凑合”到“更讲究”的升级史。早些时候，人们会尝试植物油、动物脂这类替代物；后来也有人用凡士林类或按摩油类；到了现代，产品才开始认真重视舒适度、清洁感和兼容性。

00:24 - 00:35  
真正要记住的，其实就三类。第一类是水基，清爽、好洗，也更适合大多数新手。第二类是硅基，特点是更持久，遇水也不容易马上失效。第三类是油基，顺滑感强，但如果要配乳胶安全套，就不建议选它。

00:35 - 00:40  
如果你不知道怎么选，记住这句就够了：第一次尝试先看水基，想更持久再看硅基，要配乳胶安全套就避开油基。

00:40 - 00:42  
懂类型，比跟风买最火那瓶更重要。

## 屏幕重点词

- 润滑液 ≠ 只在尴尬时才用
- 从凑合到讲究
- 三大类: 水基 / 硅基 / 油基
- 先舒服，再顺滑，最后看兼容
